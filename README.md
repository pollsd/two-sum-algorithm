Practicing two sum algorithms by brute force then optimizing it. 
# Two Sum Algorithm (Python)

## 📌 Problem
Given an array of integers, return indices of two numbers such that they add up to a target.

## 🚀 Approach
- Used hashmap for O(n) time complexity
- Avoided nested loops

## 🧠 Complexity
- Time: O(n)
- Space: O(n)
